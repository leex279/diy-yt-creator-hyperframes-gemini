What is HyperFrames? Video, with HTML.

Just an HTML file plus GSAP. To start, npx hyperframes init. No build step. No config.

Three rules to remember. Number one. data-start tells the renderer when each clip enters and exits. Number two. class equals clip controls visibility. Number three. Register your GSAP timeline on window dot double-underscore timelines.

Then npx hyperframes preview to view in the browser. npx hyperframes render to ship an MP4. Deterministic. No randomness. No clocks. No network at render time.
