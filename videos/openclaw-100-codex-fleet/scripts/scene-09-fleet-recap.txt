Back in, and here's how all of it stacks together. Claw-sweeper at the top, closing the dead issues. Crab-box in the middle, leasing the machines that run the work. Claw-patch underneath, slicing the code into reviewable units. Codex sitting across all three, doing the actual reading and writing. Vercel's deep-sec, released two weeks before this post, handling the security side, with Codex configured for security covering whatever deep-sec doesn't.

That's it. That's the hundred-agent fleet. Three thin, well-scoped tools, plus Codex, plus deep-sec. Composable, not magical.
