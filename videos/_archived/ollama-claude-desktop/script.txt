Ollama just shipped Claude Desktop support. The launch tweet exploded. 139 thousand views in hours.

Here is what just unlocked. The Claude desktop app, with any Ollama Cloud model. Open source models, frontier scale parameter counts, all running through Anthropic's interface.

First, export your Ollama API key. Then run, ollama launch claude-desktop. Reload the app and the model picker now shows Ollama models.

Switch to Claude Cowork. Pick a model from the dropdown. Let it handle real work, autonomously. Organizing files, talking to your local apps. Agentic tasks with the model you chose.

Same thing in Claude Code. Same dropdown. Same Ollama Cloud lineup. The launch demo shipped a snake game graphics upgrade plan, running on Kimi.

The lineup is deep. Kimi. GPT-OSS. Qwen. Plus Devstral, Ministral, GLM, and Minimax. Everything in Ollama Cloud, discovered automatically.

Two limits to know. Web search is not supported yet. Extensions are not supported yet. Subagents inherit your current model.

Full setup walkthrough is at docs.ollama.com slash integrations slash claude-desktop.
