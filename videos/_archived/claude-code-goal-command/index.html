How /goal works
Two models. One loop.
1
State a completion condition
e.g. "all tests pass and lint is clean"
2
Claude takes a turn
Edits files. Runs tools. Writes code.
3
Haiku judges: yes / no
Reads transcript. Returns short reason.
4
Yes auto-clears the goal
Writes "achieved." in the transcript.
