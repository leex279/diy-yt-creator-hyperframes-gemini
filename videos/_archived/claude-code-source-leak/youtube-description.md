- March 31, 2026: `@anthropic-ai/claude-code` v2.1.88 shipped to npm with a `.map` source-map file inside the bundle
- 513,000 lines of unobfuscated TypeScript across 1,906 files exposed — root cause was a missing `*.map` entry in `.npmignore`
- 41,500+ GitHub forks landed within hours of disclosure (security researcher Chaofan Shou tweeted the URL at 04:23 UTC)
