## What is OpenClaw?

OpenClaw is a free, open-source AI agent framework (formerly Clawdbot/Moltbot). It's a self-hosted agent runtime and message router that acts as a personal AI assistant running on your own infrastructure. Unlike ChatGPT or Claude chatbots, OpenClaw can actually *perform tasks* — read/write files, run shell commands, browse websites, send emails, control APIs, and automate workflows.

**Key technical facts**:
- Long-running Node.js service connecting chat platforms to an AI agent
- Model-agnostic: works with OpenAI, Anthropic (Claude), Google (Gemini), or local models
- 100+ built-in AgentSkills for expanding functionality
- Persistent memory via local Markdown documents
- 50+ third-party integrations (chat, productivity, smart home, music)
- Messaging: Telegram, WhatsApp, Slack, Discord, iMessage, Signal, Microsoft Teams, Google Chat
- Fully open-source, privacy-first, self-hosted

**What it can do**:
- Triage emails, draft replies, manage calendars, send daily summaries
- Run tests, debug code, deploy updates, monitor repositories
- Browser automation, form-filling, web scraping
- Smart home control (lighting, temperature, music, security)
- Social media scheduling, content creation
- Image generation, audio content creation
