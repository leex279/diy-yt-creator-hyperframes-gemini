Anthropic just shipped agent view in Claude Code. One list of every session you have running. Available today as a research preview.

Before this, you juggled terminal tabs, tee-mux grids, and a mental ledger of who needed you next. Now there is one place. 3 modes. 5 plans.

First, see everything. Press the left arrow or run the claude agents command. Every row shows which sessions need you, which are working, and which are done. Second, peek and reply. Select a session to read the last turn. If it is waiting on you, answer inline. Third, background anything. Use slash B G to push a running session into the view. Or claude dash dash B G to start a fresh one.

Here is how developers are using it. Dispatch several ideas at once and come back to pull requests ready for review. Babysit long running jobs that show their next run time right in the list. Bounce between sessions with arrow keys without losing context. Scan status indicators to see what shipped.

Agent view is available today on Pro, Max, Team, Enterprise, and Claude API plans. Opt in by running claude agents. If you want to learn more about AI, check out the dynamous dot A I community.
