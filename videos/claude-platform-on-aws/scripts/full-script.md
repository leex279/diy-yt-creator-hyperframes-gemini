# Anthropic just put the full Claude API on AWS

## Scene 1: The lag every AWS dev knows

Every AWS dev shipping with Claude knows this pain. Bedrock gets the new features WEEKS late.

<!-- Receipt: https://claude.com/blog/claude-platform-on-aws -->

## Scene 2: But today

But today, that lag is gone. Anthropic just shipped the full Claude Platform on AWS. Generally available today.

<!-- Receipt: https://aws.amazon.com/about-aws/whats-new/2026/05/claude-platform-aws/ -->

## Scene 3: Not a Bedrock rebrand

And here's the part most people will miss. This is not Bedrock with a new name. Bedrock is the data-resident subset. Platform-on-AWS is the full native API. Day zero.

<!-- Receipt: https://claude.com/blog/claude-platform-on-aws -->

## Scene 4: 8 betas, day zero

Which means the entire beta surface lands on AWS today. Managed Agents. Advisor. Skills. Files. MCP connector. Code execution. Web search. And the Claude Console, on AWS for the first time. The prompt improver, the generator, the evals UI.

<!-- Receipt: https://claude.com/blog/claude-platform-on-aws -->

## Scene 5: Through your AWS account

And here's how it ships through your AWS account. The endpoint lives on aws-external-anthropic.region.api.aws. Auth is through Identity Access Management. Every call is a CloudTrail event. Billing rides on AWS Marketplace, retiring against your existing commitment.

<!-- Receipt: https://aws.amazon.com/blogs/machine-learning/introducing-claude-platform-on-aws-anthropics-native-platform-through-your-aws-account/ -->

## Scene 6: 17 regions at launch

Plus, it's available in 18 AWS regions at GA. That's broader than most Claude routes at launch.

<!-- Receipt: https://aws.amazon.com/blogs/machine-learning/introducing-claude-platform-on-aws-anthropics-native-platform-through-your-aws-account/ -->

## Scene 7: Three customers already shipping

Three named customers are already shipping on it. ReliaQuest in cybersecurity. OpenRouter routing through AWS Identity Access Management. And Emergent, with day-one access to every new model.

<!-- Receipt: https://claude.com/blog/claude-platform-on-aws -->

## Scene 8: So when do you pick which

So when do you pick which? If you want every beta from day one, that's Platform-on-AWS. If your data must stay inside AWS, that's Bedrock.
