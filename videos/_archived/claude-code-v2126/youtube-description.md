Key Changes in This Release:
- Gateway model picker — choose which model serves your requests directly from the claude.ai web UI, with support for Anthropic-compatible gateway configurations
- claude project purge — new CLI command to delete all Claude Code data for a project without starting a fresh session from scratch
- --dangerously-skip-permissions flag — bypass all permission prompts for CI pipelines and headless/unattended automation use cases
- OAuth URL paste in terminal — complete browser-based authentication without leaving your shell; paste the URL directly into the prompt
- Windows PowerShell promoted to primary shell — Claude Code now auto-detects and uses PowerShell as the first-class shell on Windows
- Image paste fix — paste screenshots and diagrams directly into Claude Code input; images now reach the model correctly
- MCP & Remote Control — claude.ai connectors unblocked from needs-auth state, retry attempts now show results, failure notifications include error reasons
